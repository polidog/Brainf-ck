元々は池袋バイナリ勉強会に参加したときにつくってたんですが、  
なんかjojo言語とかプログラミング言語KQとかいろいろと面白いものがあったので、

動作環境
------------
php5.4以上  
みなさん、php5.4使いましょう＾＾


動かし方
------------
ジョジョ言語

 	php example/jojo.php jojo/hello.jojo 

プログラミング言語KQ

	php example/kq.php kq/hello.qk

ニャルラトホテプ言語

	php example/nyaruko.php nyaruko/hello.nc
